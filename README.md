# 🚀 SERRATEC 2026-01: AVALIAÇÃO FINAL
> Disciplina: Lógica de Programação

Este repositório contém o projeto final desenvolvido em **Portugol (Portugol Studio)**. O sistema foca em aplicar conceitos fundamentais de lógica, como estruturas condicionais, laços de repetição e manipulação de vetores/matrizes.

## 📋 Sobre o Projeto
O projeto consiste na reunião de exercicios propostos e suas resoluções por cada integrante.

## ✨ Funcionalidades Principais aplicaveis
- [ ] Cadastro de [Ex: Clientes/Produtos]
- [ ] Listagem e busca de dados
- [ ] Cálculos automáticos de [Ex: Médias/Totais/Impostos]
- [ ] Relatórios de saída no console
- [ ] Validação de entradas de dados

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** Portugol
- **Ambiente de Desenvolvimento:** [Portugol Studio](http://lite.acad.univali.br/portugol/)
- **Gestão:** GitHub

## 🚀 Como Executar
1. Certifique-se de ter o **Portugol Studio** instalado em sua máquina ou WEB.
2. Faça o download do arquivo `.por` deste repositório.
3. Abra o Portugol Studio.
4. Vá em `Arquivo > Abrir` e selecione o ficheiro descarregado.
5. Pressione a tecla `F9` ou clique no botão de "Play" para executar o programa.

## 🧠 Lógica e Estrutura
Para a construção deste sistema, o grupo utilizou:
- **Estruturas de Decisão:** `escolha-caso` para o menu principal e `se-senao` para validações.
- **Laços de Repetição:** `enquanto` para manter o programa rodando até que o utilizador decida sair.
- **Armazenamento:** `vetores` para persistência temporária dos dados durante a execução.
