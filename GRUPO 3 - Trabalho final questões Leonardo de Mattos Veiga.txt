Questão 3 - Separar números pares e ímpares de um vetor
 Descrição: Crie um programa que leia 10 números inteiros e armazene em um vetor. Depois, separe em dois vetores, um de impares e outro de pares, depois exiba os números ímpares e os
pares separadamente.

*Essa foi a questão que deu mais trabalho, seguindo o enunciado essa é a resposta que encontrei:
*OBS = Coloquei uma resposta abaixo dessa por conta de uma dificuldade de entendimento com o enunciado, onde a resposta ficaria mais simples, e veio um entendimento melhor de vetores após as aulas de vetores e matrizes.

programa
{
    funcao inicio()
    {
        inteiro numeros[10]
        inteiro pares[10]
        inteiro impares[10]
        inteiro i
        inteiro par = 0
        inteiro impar = 0

        para (i = 0; i < 10; i = i + 1)
        {
            escreva("Digite um numero: ")
            leia(numeros[i])

            se (numeros[i] % 2 == 0)
            {
                pares[par] = numeros[i]
                par = par + 1
            }
            senao
            {
                impares[impar] = numeros[i]
                impar = impar + 1
            }
        }

        escreva("\nNumeros pares:\n")

        para (i = 0; i < par; i = i + 1)
        {
            escreva(pares[i], "\n")
        }

        escreva("\nNumeros impares:\n")

        para (i = 0; i < impar; i = i + 1)
        {
            escreva(impares[i], "\n")
        }
    }
}

Questão 16 - Contar Números Negativos em um Vetor .
Descrição: Escreva uma função que receba um vetor de 10 números e retorne quantos são negativos. 

programa
{
    inteiro vetor[10]

    funcao inteiro contarNegativos()
    {
        inteiro i
        inteiro contador
        contador = 0

        para(i = 0; i < 10; i = i + 1)
        {
            se(vetor[i] < 0)
            {
                contador = contador + 1
            }
        }

        retorne contador
    }

    funcao inicio()
    {
        inteiro i
        inteiro resultado

        para(i = 0; i < 10; i = i + 1)
        {
            escreva("Digite um numero: ")
            leia(vetor[i])
        }

        resultado = contarNegativos()

        escreva("\nQuantidade de numeros negativos: ", resultado)
    }
}
